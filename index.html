Cum functioneaza

Monte Carlo — Geometric Brownian Motion

Programul simuleaza 10.000 de scenarii posibile pentru pretul unei actiuni pe baza volatilitatii si trendului istoric din ultimul an.

Pret(t+1) = Pret(t) × exp(drift + sigma × Z)

Unde drift este tendinta medie zilnica, sigma este volatilitatea zilnica, iar Z este un numar aleator din distributia normala.

Analiza Sentiment

Programul descarca stiri recente din 3 surse — Yahoo Finance, Reuters RSS si Google News — si le analizeaza cu un algoritm VADER (Valence Aware Dictionary and sEntiment Reasoner) implementat local, fara API key.

Sentimentul influenteaza parametrii GBM: un sentiment pozitiv creste drift-ul, iar un sentiment negativ sau extrem creste volatilitatea.

Cei 7 factori externi

Stirile sunt clasificate automat in 7 categorii: geopolitic, inflatie/dobanzi, crize financiare, pandemii, tarife comerciale, alegeri/politica si stiri despre companie. Fiecare factor primeste un scor de la -1.0 (bearish) la +1.0 (bullish).

Date

Preturile istorice vin din Yahoo Finance via un proxy public. Stirile vin din RSS feeds publice, fara cont sau API key necesar.

⚠️ DISCLAIMER: Aceasta aplicatie este o simulare matematica bazata pe date istorice si analiza de sentiment automata. Nu reprezinta sfat financiar. Pietele sunt influentate de factori imposibil de modelat complet. Foloseste cu discernamant.
