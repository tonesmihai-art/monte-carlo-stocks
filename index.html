Sigma: i
📊 Sigma + GARCH(1,1) Primul numar = sigma statica (medie istorica). Al doilea = sigma GARCH conditionata — cat e volatilitatea chiar acum, estimata din ultimele 60 de zile.
🟢 GARCH < sigma — regim calm
🟡 GARCH ≈ sigma — regim normal
🔴 GARCH > sigma — regim agitat
Simularea porneste cu GARCH actual si revine treptat la sigma istorica pe masura ce trece timpul. Influenteaza: dispersia traiectoriilor
—
Vol. anuala: i
📅 Volatilitate anuala + Persistenta GARCH Primul numar = sigma × √252. Al doilea (pers.) = α+β din GARCH — cat de "lipicioasa" e volatilitatea.
Pers. < 85% — socurile dispar repede
Pers. 85–95% — volatilitate persistenta
Pers. > 95% — soc dureaza saptamani
Influenteaza: viteza de revenire la vol. normala

Drift: i
📈 Drift — Tendinta zilnica Directia medie a pretului per zi, calculata din istoricul unui an. Drift pozitiv = trend ascendent. Influenteaza direct unde se concentreaza simularile.
> 0 — tendinta de crestere
≈ 0 — piata laterala
< 0 — tendinta de scadere
Influenteaza: media preturilor simulate

MA60: i
〰️ Media mobila 60 zile Media pretului din ultimele 60 de zile. Procentul arata cat de departe e pretul curent fata de aceasta medie (mean reversion).
Pret > MA → presiune de scadere
Pret < MA → presiune de crestere
Deviatia mare fata de MA ajusteaza drift-ul in simularea AI. Influenteaza: mean reversion in GBM

Cum functioneaza

Monte Carlo — Geometric Brownian Motion

Programul simuleaza 30.000 de scenarii posibile pentru pretul unei actiuni pe baza volatilitatii si trendului istoric din ultimul an.

Pret(t+1) = Pret(t) × exp(drift + sigma × Z)

Unde drift este tendinta medie zilnica, sigma este volatilitatea zilnica, iar Z este un numar aleator din distributia normala.

Detectie sector + ponderi dinamice

Aplicatia detecteaza automat sectorul actiunii (Technology, Energy, Financial Services etc.) si aplica ponderi diferite celor 7 factori. De exemplu, pentru o companie de energie, geopolitica si tarifele comerciale conteaza mai mult decat pentru o companie tech.

VIX — Indicele fricii

VIX masoara volatilitatea implicita a pietei. Cand VIX depaseste 25, sigma din simulare creste automat pentru a reflecta incertitudinea crescuta.

Analiza Sentiment

Stirile sunt descarcate din Yahoo Finance, Reuters RSS si Google News, analizate cu VADER implementat local (fara API key) si clasificate in 7 factori, fiecare ponderat in functie de sectorul actiunii.
